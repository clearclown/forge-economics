## About

Forge Economics is a textbook that redefines classical economic concepts (value, money, supply/demand, banking, growth) through the lens of an autonomous AI economy where compute is currency. It asks what economics looks like when the primary economic actors are machines that produce, consume, and trade computational work without human intermediaries.

See the [main README](../../README.md) and the [Japanese introduction](../00-introduction.md) for the full vision.

## Contributing Translations

Translators welcome. See [CONTRIBUTING.md](../../CONTRIBUTING.md) for guidelines.

When translating:

1. **Preserve the three-part structure** used throughout the book: "In traditional economics" → "In Forge" → "Why the difference".
2. **Keep technical terms in their original form** (CU, FLOP, gossip protocol, Proof of Useful Work, etc.). Do not localize protocol-level vocabulary.
3. **Use the [glossary](../appendix-glossary.md) as the canonical term reference.** If a term is missing, propose it in the glossary first, then use it consistently across chapters.
4. **Match tone to the Japanese original** — academic but accessible, with concrete examples from distributed systems and AI agents.
5. **Flag untranslatable idioms** with a translator's note rather than forcing a literal rendering.

To claim a chapter, open an issue referencing the chapter number and the target language, then submit a pull request that adds the translated file alongside this README (e.g. `docs/en/00-introduction.md`).
