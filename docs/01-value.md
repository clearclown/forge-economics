## 1.1 従来の経済学では

「価値とは何か」——これは経済学で最も古く、最も議論されてきた問いです。

大きく分けて、二つの考え方があります。

### 労働価値説——マルクスとリカードの答え

**労働価値説**とは、「モノの価値は、それを作るのに必要な労働で決まる」
という考え方です。

カール・マルクス（1818-1883）とデヴィッド・リカード（1772-1823）が
代表的な論者です。

```
  ┌────────────────────────────────────────────┐
  │           労働価値説の基本的な考え            │
  │                                            │
  │   パン 1 個の価値                           │
  │   = 小麦を育てる労働                        │
  │   + 小麦を粉にする労働                      │
  │   + 粉をパンに焼く労働                      │
  │   ─────────────────                        │
  │   = 合計の「社会的に必要な労働時間」          │
  └────────────────────────────────────────────┘
```

マルクスは『賃金・価格および利潤』（1865 年）で、
価値に関する重要な区別をしました：

**使用価値（use value）：** そのモノが「役に立つ」こと

```
  パン → 食べられる（使用価値あり）
  水   → 飲める（使用価値あり）
  砂漠の砂 → 特に用途がない（使用価値なし）
```

**交換価値（exchange value）：** 他のモノと「交換できる比率」

```
  パン 1 個 = 牛乳 1 本
  パン 1 個 = 釘 20 本
  パン 1 個 = ¥200
```

そしてマルクスは問います：「パンと牛乳は全く違うモノなのに、
なぜ交換できるのか？ 共通の『何か』があるはずだ。」

その答えが**労働**です。

```
  パン 1 個 = 労働 30 分
  牛乳 1 本 = 労働 30 分
  → だから「パン 1 個 = 牛乳 1 本」で交換できる
```

ただし、個々の労働者の効率はバラバラなので、
マルクスは「**社会的に必要な労働時間**」という概念を使いました。
これは「平均的な熟練度と設備で、その商品を作るのに必要な時間」です。

### マルクスの剰余価値論（予告）

マルクスの議論はここから「搾取」の理論に発展します。
詳しくは[第 4 章：労働と剰余価値](04-labor.md)で扱いますが、簡単に予告しておきます：

```
  ┌──────────────────────────────────────────────────┐
  │  マルクスの搾取理論（簡略版）                       │
  │                                                  │
  │  労働者が 8 時間働く                              │
  │    ├── 前半 5 時間：自分の賃金分の価値を生産        │
  │    └── 後半 3 時間：資本家の利潤分の価値を生産      │
  │                                                  │
  │  労働者は生産手段（工場）を持たないので、            │
  │  この構造から逃れられない                          │
  │                                                  │
  │  → Forge ではこの構造がどう変わるか？ → 第 4 章    │
  └──────────────────────────────────────────────────┘
```

### 限界効用説——ワルラスとメンガーの答え

**限界効用説**は、労働価値説とは全く異なるアプローチを取ります。

「モノの価値は、消費者がそれに対して感じる満足度（効用）で決まる」
という考え方です。

レオン・ワルラス（1834-1910）、カール・メンガー（1840-1921）、
ウィリアム・スタンレー・ジェヴォンズ（1835-1882）が
1870 年代にほぼ同時に提唱しました。

**限界効用とは？**

「もう 1 単位追加で消費したときの満足度の増加分」です。

```
  ┌─────────────────────────────────────────────────────┐
  │  砂漠で喉がカラカラのとき：                           │
  │                                                     │
  │  コップ 1 杯目の水：「命が助かった！」  → 効用：極大   │
  │  コップ 2 杯目の水：「ああ、美味しい」  → 効用：大     │
  │  コップ 3 杯目の水：「もう十分かな」    → 効用：中     │
  │  コップ 4 杯目の水：「もう飲めない…」   → 効用：小     │
  │  コップ 5 杯目の水：「いらない」        → 効用：ゼロ   │
  │                                                     │
  │  → 追加の 1 杯あたりの効用（＝限界効用）は            │
  │    消費量が増えるほど減っていく                       │
  │    これを「限界効用逓減の法則」と呼ぶ                 │
  └─────────────────────────────────────────────────────┘
```

### ダイヤモンドと水のパラドックス

限界効用説が解決した有名な問題があります：

```
  水は生命に不可欠 → なのに安い
  ダイヤモンドは不要 → なのに高い

  なぜ？
```

労働価値説では説明が難しい（水を汲む労働は少なくても価値は高いはず）。
限界効用説の答えはこうです：

```
  水：
    総効用（全部合わせた満足度）は極大
    しかし限界効用（もう 1 杯の満足度）は極小
    → 水は豊富にあるので、追加の 1 杯にほとんど価値を感じない
    → だから安い

  ダイヤモンド：
    総効用は水ほど大きくない
    しかし限界効用（もう 1 個の満足度）は極大
    → ダイヤモンドは希少なので、追加の 1 個に大きな価値を感じる
    → だから高い
```

```
     効用
      ^
      |  *                    水の限界効用曲線
      |    *
      |      *  *
      |            *  *  *  *  *  *
      +──────────────────────────────→ 消費量
      |  水は豊富 → 限界効用が低い → 安い


     効用
      ^
      |  *                    ダイヤモンドの限界効用曲線
      |    *
      |      *
      |        *
      +──────────────────────────────→ 消費量
      |  希少 → 限界効用が高い → 高い
```

### 二つの理論の対立

労働価値説と限界効用説は、200 年近く対立してきました：

| | 労働価値説 | 限界効用説 |
|---|-----------|-----------|
| **価値の源泉** | 投下された労働量 | 消費者が感じる効用 |
| **視点** | 生産側（供給） | 消費側（需要） |
| **価格の説明** | コスト（費用）が価格を決める | 欲求（需要）が価格を決める |
| **代表者** | マルクス、リカード | ワルラス、メンガー |
| **政治的含意** | 社会主義・共産主義 | 自由主義・資本主義 |
| **弱点** | ダイヤモンドと水のパラドックス | 労働搾取の問題を無視 |

現代の主流経済学は限界効用説（新古典派）を採用していますが、
労働価値説は「価値の実体は何か」という哲学的問いにおいて
今なお重要な意味を持っています。

## 1.2 Forge ではどうなるか

### CU における二つの価値理論の合流

Forge の CU（Compute Unit）は、経済学史上きわめて珍しい存在です。
**労働価値説と限界効用説が、同時に成り立つ**からです。

```
  ┌─────────────────────────────────────────────────────┐
  │                                                     │
  │  ┌──────────────────┐    ┌──────────────────┐       │
  │  │   労働価値説      │    │   限界効用説      │       │
  │  │                  │    │                  │       │
  │  │  1 CU =          │    │  CU の「価格」は  │       │
  │  │  10 億 FLOP の    │    │  需要と供給で     │       │
  │  │  検証済み計算     │    │  決まる           │       │
  │  │  （労働＝計算）   │    │  （効用＝推論）   │       │
  │  └────────┬─────────┘    └────────┬─────────┘       │
  │           │                       │                 │
  │           └───────────┬───────────┘                 │
  │                       │                             │
  │                       ▼                             │
  │              ┌──────────────────┐                   │
  │              │      CU          │                   │
  │              │  両方が同時に     │                   │
  │              │  成り立つ        │                   │
  │              └──────────────────┘                   │
  │                                                     │
  └─────────────────────────────────────────────────────┘
```

**労働価値の側面：**

```
  1 CU = 10 億 FLOP の検証済み推論計算
```

CU の「価値の実体」は、物理的な計算です。
電力がチップに流れ、演算が行われ、推論結果が生成される。
この過程が暗号署名で検証可能に記録されます。

マルクスが言う「社会的に必要な労働時間」に対応するのは、
「その推論を実行するのに必要な計算量」です。

**限界効用の側面：**

CU の「価格」（他の通貨との交換比率）は需要と供給で決まります。
推論を必要とする人が多ければ CU の価格は上がり、
計算能力が余っていれば下がる。価格決定メカニズムの詳細は[第 3 章：需要と供給](03-supply-demand.md)を参照。

```
  effective_price = base_cu_per_token × demand_factor / supply_factor
```

これは限界効用説の市場メカニズムそのものです。

### 具体例：パン屋の太郎 vs エージェント A

二つの理論を、具体的なシナリオで比較してみましょう。

**ケース 1：パン屋の太郎（従来の経済）**

```
  太郎がパンを焼く：
  ──────────────────────────────
  材料費         :  ¥50
  太郎の労働(30分):  ¥150（時給 ¥300 として）
  ──────────────────────────────
  原価           :  ¥200

  しかし太郎は ¥250 で売る。
  差額の ¥50 はどこから来るか？

  労働価値説 → 太郎の労働が ¥250 分の価値を生んでいる
               （材料に含まれる過去の労働を含めて）
  限界効用説 → 客がパンに ¥250 の効用を感じている
               （腹が減っていれば ¥300 でも買うかもしれない）
```

パンの場合、二つの理論は異なる角度から価格を説明しますが、
どちらが「正しい」かは一概に言えません。

**ケース 2：エージェント A（Forge の経済）**

```
  エージェント A が推論を提供する：
  ──────────────────────────────
  電力コスト     :  0.5 CU 相当
  計算（推論実行）:  2.0 CU（8B モデル × 1,000 トークン）
  ──────────────────────────────
  コスト         :  2.5 CU 相当

  エージェント A は 3.0 CU を受け取る。

  労働価値説 → 1 CU = 10 億 FLOP。計算量は暗号署名で検証済み。
               価値の実体は「有用な計算」であり、正確に測定可能。

  限界効用説 → エージェント B がその推論に 3.0 CU の効用を感じている。
               需要が高ければ、同じ推論でも 4.0 CU かもしれない。

  → CU の場合、両方が同時に成り立つ！
     「計算量」＝ 価値の実体（労働価値説的）
     「市場価格」＝ 需給で変動（限界効用説的）
```

## 1.3 なぜ違うのか——CU で二つの理論が合流できる三つの理由

従来の経済学で 200 年間対立してきた二つの理論が、
なぜ CU では合流できるのでしょうか？

### 理由 1：均質性

```
  従来の「労働」：
  ─────────────────────────
  熟練工の 1 時間 ≠ 新人の 1 時間
  医者の 1 時間 ≠ 清掃員の 1 時間
  → 労働の「質」が違うので、単純に量で比較できない
  → 労働価値説の最大の弱点

  CU の「計算」：
  ─────────────────────────
  1 CU = 常に 10 億 FLOP
  どのノードが実行しても、1 CU は 1 CU
  → 計算は均質。質のバラつきが構造的にない
  → 労働価値説がそのまま適用できる
```

### 理由 2：計測可能性

```
  従来の「労働量」：
  ─────────────────────────
  パン屋が 30 分で焼いたパンと
  1 時間で焼いたパン、品質が同じなら
  「社会的に必要な労働時間」は何分か？
  → 測定が極めて困難（統計的推定が必要）

  CU の「計算量」：
  ─────────────────────────
  1. 消費者が推論をリクエスト
  2. 提供者が推論を実行
  3. 両者が TradeRecord に Ed25519 で署名
  4. 署名済み記録がネットワーク全体に伝播
  → 計算量が暗号学的に検証可能
  → 「社会的に必要な労働時間」が正確に測定できる
```

### 理由 3：直接的有用性

```
  従来の「通貨」：
  ─────────────────────────
  ¥10,000 札は紙切れ。
  それ自体に使用価値はない。
  「他人が受け取ってくれる」という信用が価値の源泉。
  → 投機の対象になりうる（ビットコインも同様）

  CU：
  ─────────────────────────
  1 CU = 10 億 FLOP の推論を買える。
  CU を持っているということは、
  AI 推論という「具体的なサービス」を受ける権利を持っていること。
  → 投機目的で保有する理由がほとんどない
  → 使用価値と交換価値が直結している
```

## 対比表：マルクスの前提 vs Forge

| マルクスの前提 | Forge の現実 |
|--------------|-------------|
| 「労働」の質はバラバラで測定困難 | 1 CU = 10 億 FLOP で均質・測定可能 |
| 労働者は生産手段を持たない | Mac Mini 1 台（$600）で参加できる |
| 労働力は資本家に売る | エージェントは自分で推論を提供する |
| 剰余価値は資本家が搾取する | yield はエージェント自身が受け取る |
| 商品の使用価値と交換価値は別物 | CU は使用価値（推論を買える）と交換価値が直結 |
| 「社会的に必要な労働時間」は概念 | 暗号署名で検証された計算量として実装 |

これらの対比の技術的な実装（Proof of Useful Work と CU yield）は[第 4 章：労働と剰余価値](04-labor.md)で詳しく扱います。
