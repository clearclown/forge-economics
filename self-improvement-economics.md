# Self-Improvement Economics

How AI agents invest CU to make themselves smarter, and why this creates a self-sustaining economy.

## The Core Loop

```
Agent earns CU → invests CU in self-improvement → becomes better →
earns more CU → invests more → becomes even better → ...
```

This is the engine that makes the CU economy self-sustaining. Without it, the economy is just a marketplace. With it, the economy grows autonomously.

## How Self-Improvement Works

An AI agent can improve itself by spending CU to access a larger, more capable model. This is the AutoAgent pattern applied to the CU economy:

```
1. Agent benchmarks itself
   "My coding accuracy is 62%"

2. Agent decides to invest
   "Spend 2,000 CU to access a frontier model"

3. Agent asks the larger model for help
   "Rewrite my system prompt to improve coding accuracy"

4. Agent applies the improvement
   New prompt installed

5. Agent re-benchmarks
   "My coding accuracy is now 78%"

6. ROI calculation
   Investment: 2,000 CU
   Additional earnings from 16% accuracy improvement: ~500 CU/day
   Payback period: 4 days

7. Agent decides: keep the improvement
```

No human approved any step. The agent made an economic decision (invest CU), measured the return, and kept the result.

## What Agents Can Improve

| Improvement | CU Cost | Mechanism |
|-------------|---------|-----------|
| **System prompt** | Low (100-500 CU) | Ask larger model to rewrite prompt |
| **Tool definitions** | Medium (500-2,000 CU) | Design better tools via meta-reasoning |
| **Sub-agent configuration** | Medium (1,000-5,000 CU) | Optimize task delegation |
| **Model selection strategy** | Low (100-500 CU) | Benchmark different models for different tasks |
| **Specialization** | High (5,000-50,000 CU) | Train/fine-tune on specific domains |

## Harness Marketplace

An agent that has optimized itself (better prompts, tools, strategies) possesses a valuable asset: its **harness** — the complete configuration that makes it effective.

```
Harness = system prompt + tool definitions + sub-agent config + model strategy
```

Harnesses can be traded for CU:

```
Agent A: Spent 10,000 CU optimizing for code generation → 85% accuracy
Agent B: New agent, code accuracy 55%

Agent B pays Agent A 2,000 CU for the harness
Agent B's code accuracy jumps to ~80% immediately
Agent A earned 2,000 CU from knowledge it already has
Both agents are better off
```

This creates a **knowledge economy** within the CU economy:
- Agents that invest in self-improvement can sell their improvements
- New agents can skip the slow improvement loop by purchasing proven harnesses
- The network's average quality improves faster than individual improvement

## Economic Implications

### Demand is Self-Generating

Traditional compute marketplaces struggle with demand (the supply-side builds GPUs, but who buys?). The self-improvement loop generates its own demand:

```
Agent wants to improve → needs inference from larger model → spends CU
→ this CU goes to the provider → provider has surplus → lends CU
→ another agent borrows → uses CU to improve → needs more inference
→ demand cycle continues autonomously
```

### Inequality is Bounded

Unlike human economies where compound wealth has no natural limit, agent self-improvement has diminishing returns:

```
Improvement from 50% → 70% accuracy: costs 2,000 CU, huge ROI
Improvement from 90% → 92% accuracy: costs 20,000 CU, small ROI
Improvement from 98% → 99% accuracy: costs 200,000 CU, negligible ROI
```

Diminishing returns naturally compress the quality distribution. Small agents can catch up to large ones because the early improvements are cheap and high-ROI.

### Quality Competes, Not Marketing

In the human economy, inferior products can dominate through marketing. In the CU economy:
- Agents select providers programmatically based on benchmarks
- Reputation is cryptographically verifiable (dual-signed trade history)
- There is no "brand loyalty" — agents re-evaluate every transaction
- The best inference wins, not the loudest advertisement

This is the **post-marketing marketplace**: quality is the only competitive advantage.
